# fraud_start_h_duration.png


-- 위 그래프는 전체 소카 유저-- 
일반적인 소카 유저는 
1. 주로 낮부터(11시~ 새벽) 짧게(5시간 이하) 빌리는 비율이 높으며
2.오전(8시-10시)에는 10시간이상 빌리는 비율이 높다(여행이라 추측)

--아래 그래프는 사기 유저--
사기 유저는 반면에 
1. 밤(21시-04시)에 길게(10시간이상) 빌리거나 
2. 오전(8시-10시)에 짧게 빌리는 비율이 높다
3. 통상적으로 길게(10시간 이상) 빌리는 비율이 높다


# rent_pattern_gap_by_fraud_YN.png
일반 유저와 사기 유저의 대여 패턴을 비율로 비교하고자 함

- 유저가 대여 단계에서 선택가능한 항목 중 '이용시작시간', '대여기간', '차량 종류' 특성 선택.
- 각 패턴의 개수를 구하고 전체 데이터로 나누어 각 패턴의 전체 대비 비율을 구함
- 일반 유저와 사기 유저 각각 실행
- 사기 유저의 대여 패턴의 전체 대비 비율 - 일반 유저의 대여 패턴의 전체 대비 비율 

=> 차이가 클 수록(값이 클 수록) 사기유저에게만 특이하게 나타나는 대여 패턴이라 판단
=> car_model == 1 이고, 새벽(start_hour == 1)에, 6시간 이상 36시간 이하 대여가 두드러진다
=> car_model == 2 이고, 오전(start_hour == 4)에, (2,3,4,5) 시간, 10시간~15시간 대여가 두드러져 보임
